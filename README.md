# Sales Data Analysis Project

## Overview
This project analyses sales data using SQL and Python to uncover trends in revenue, customer behaviour, and product performance.

## Tools Used
- SQL (SQLite)
- Python (Pandas, Matplotlib)

## Key Insights
- Revenue peaked in March (£2800), showing an upward trend
- Phones were the highest revenue-generating product
- The UK contributed ~80% of total revenue
- Bob and Alice were the highest-value customers
- Furniture products underperformed compared to electronics

## Files
- sales_data.csv: Cleaned dataset
- analysis.ipynb: Python analysis and visualisations
